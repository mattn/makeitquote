# nostr-makeitquote

makeitquote bot for nostr

## Usage

```
MAKEITQUOTE_NSEC=nsecxxxxxxxxxxxxx ./nostr-makeitquote
```

## Requirements

* TrueType Font (default: Koruri-Bold.ttf)

## Installation

```
$ go install github.com/mattn/nostr-makeitquote@latest
```
